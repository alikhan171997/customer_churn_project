# customer_churn_project
An end-to-end Machine Learning pipeline designed to analyze customer retention drivers and predict churn risks. Built using modular Python scripts for data cleaning and automated visualization, paired with an XGBoost classifier for high-accuracy predictions. Includes dashboard reporting for feature importance and churn probabilities.
