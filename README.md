# customer_churn_project
An end-to-end Machine Learning pipeline designed to analyze customer retention drivers and predict churn risks. Built using modular Python scripts for data cleaning and visualization, paired with an XGBoost classifier for high-accuracy predictions. Includes dashboard reporting for feature importance and churn probabilities.
following is the link for bi dashboard (https://drive.google.com/file/d/1_UFuB1T02caNspw0fYPxnnTfkaz6a_p1/view?usp=sharing)

Project Summary
Goal: Predict customer churn and identify key retention factors.
Tech Stack: Python, Pandas, XGBoost, Scikit-Learn, Seaborn/Matplotlib.

Key Features: 
1. Modular data pipeline with centralized EDA visualization scripts.
2. Automated image output to `visuals/` for easy dashboard integration.
3. Model optimization using hyperparameter tuning and standard metrics.
